# Supplier Evaluation & Procurement Support System

## Overview
A Django-based web application to manage and evaluate suppliers based on cost, delivery time, and quality metrics.

## Features
- Add and manage suppliers
- Sort by cost, delivery time, and quality
- Automatic best supplier selection using scoring logic
- Admin panel support

## Tech Stack
- Django
- SQLite
- HTML

## How to Run
1. Clone the repo
2. Install Django: pip install django
3. Run migrations: python manage.py migrate
4. Start server: python manage.py runserver
